# Monitoreo de manglar en la CGSM
Pipeline GeoAI multilenguaje (Python, R, Julia) para el monitoreo dinamico de la cobertura de manglar en la Cienaga Grande de Santa Marta.

## Estructura
- `notebooks/` — Notebooks Quarto (.qmd) por fase
- `src/` — Funciones reutilizables (Python, R, Julia)
- `data/` — Datos crudos, procesados y de validacion
- `outputs/` — Figuras, mapas y tablas de resultados

## Requisitos
- Docker (contenedor sig_unal v1.11)
- Cuenta Google Earth Engine autenticada
